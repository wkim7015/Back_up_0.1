BACK-UP(V.1.0) [README]

This program is a basic light-weighted program which copies a whole given directory in to a specific directory from the input

Acknowledgement

1. Main

2. Related functions 

 					1.  address function

 					2.  copy function

					3.  remove function 

 					4.  log function

Main

 This function is to operate all available functions, and write raised errors to an error log, formatted to a txt file.

Input


directory path from which an user wants to "back-up"
log path from which a user wants to store. ( if it is not given, it will store in the given directory path)

copy function

 This function is to copy all files in a given directory. 
	if it is protected or not found in the path, then the function will raise correspond errors

Input


directory path from which an user wants to "back-up"
log path from which a user wants to store. ( if it is not given, it will store in the given directory path)

remove function

 This function is to delete the old back-up files, up to 3 weeks from the current date. If files or directories are protected by 
	admin, it will raise associated error

Input

directory path

log function

 This function has required methods for writing a log in text format style

Input

None

address function

 This function formats and delivers directory paths to required functions

Input


directory path from which an user wants to "back-up"
log path from which a user wants to store. ( if it is not given, it will store in the given directory path)
